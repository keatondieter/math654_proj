TEST01
  SPARSE_GRID_HERM_SIZE returns the number of distinct
  points in a Gauss-Hermite sparse grid.

  Note that, unlike most sparse grids, a sparse grid
  based on Gauss-Hermite points is almost entirely 
  NOT nested.

  Hence the point counts should be much higher than for a grid of
  the same level, but using rules such as Fejer2 or
  Gauss-Patterson or Newton-Cotes-Open.

  Each sparse grid is of spatial dimension DIM,
  and is made up of all product grids of levels up to LEVEL_MAX.
